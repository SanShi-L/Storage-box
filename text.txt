你需要做的：Azure 注册（一次性，约 5 分钟）
打开 https://entra.microsoft.com → 左侧「应用注册 / App registrations」→「新注册 / New registration」。
名称随便填，如 xiaoshi-shoucang。
支持的账户类型：选「任何组织目录中的账户和个人 Microsoft 账户」(Accounts in any org directory and personal Microsoft accounts)。
重定向 URI：平台选「单页应用程序 / Single-page application (SPA)」，填：

https://app.lxlsanshi.site/
注册后建议再到「身份验证 / Authentication」里把这个也加上（以防带 index.html 访问）：

https://app.lxlsanshi.site/index.html
点「注册」。
进「API 权限 / API permissions」→「添加权限」→ Microsoft Graph →「委托的权限 / Delegated」→ 搜索勾选 Files.ReadWrite.AppFolder → 添加。（个人账户用这个权限登录时会让你同意，不需要管理员）
回「概述 / Overview」，复制「应用程序(客户端) ID / Application (client) ID」。
然后
把这次改动 commit + push 部署上线后，在手机/电脑打开 App → 首页「OneDrive 云同步」→「去设置」→ 粘贴刚才那串 client_id → 点「上传到云」会弹出微软登录，登录并同意权限后即开始备份。换设备时同样填一次 client_id，点「从云恢复」即可。
